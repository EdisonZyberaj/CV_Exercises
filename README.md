# Computer Vision Exercises

This repository contains practical Computer Vision exercises implemented in Python using Jupyter Notebook and Scikit-image.

The project includes image processing, segmentation, filtering, feature extraction, and object detection techniques developed during Computer Vision course.

---

## Repository Contents

### Exercise 3
Topics covered:
- Image preprocessing
- Grayscale conversion
- Gamma correction
- Otsu thresholding
- Connected component labeling
- Region property extraction
- Balloon detection using feature analysis
- Bounding box visualization

### Exercise 4
Topics covered:
- Image filtering
- Gaussian blur
- Laplacian filtering
- Canny edge detection
- Image resizing
- Feature visualization
- Basic image statistics
